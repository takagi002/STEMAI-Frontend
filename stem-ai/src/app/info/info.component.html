Tutorial 1. Enter your Gannon email ID in the box named Gannon Email ID. ex. smith001 2. Enter your Gannon ID number in the box named Gannon ID number. ex. 123456 3. Choose your role. 3.a If you are a professor click the professor button below the Gannon ID Number input box 3.b If you are a student click the student button below the Gannon ID Number input box 4. When you click the button an email will be sent to your Gannon Email with a verification code. 5. Enter the verification code from the email in the textbox from the popup. After you entered the verification code, click enter. 6. You will automatically be relocated to the professor or student page. User Info
